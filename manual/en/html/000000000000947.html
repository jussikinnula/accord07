Valve Body and ATF Strainer Installation

Exploded View
NOTE: Refer to the exploded view as needed during the following procedures.
  1. Make sure that the ATF magnet is cleaned and installed in the torque converter housing.
  1. Install the main separator plate (A) and two dowel pins on the torque converter housing. Then install the ATF pump drive gear (B), driven gear (C), and ATF pump driven gear shaft (D). Install the ATF pump driven gear with its grooved and chamfered side facing down.
  1. Install the main valve body.
  1. Make sure the ATF pump drive gear (A) rotates smoothly in the normal operating direction, and the ATF pump driven gear shaft (B) moves smoothly in the axial and normal operating direction.
  1. If the ATF pump drive gear and ATF pump driven gear shaft do not move smoothly, loosen the main valve body bolts. Realign the ATF pump driven gear shaft, and retighten the bolts to the specified torque, then recheck. Failure to align the ATF pump driven gear shaft correctly will result in a seized ATF pump drive gear or ATF pump driven gear shaft.
  1. Make sure that the two check balls and the cooler check valve are in the main valve body, then install the cooler check valve spring in the cooler check valve.
  1. Install the servo separator plate and two dowel pins on the main valve body.
  1. Install the servo body (11 bolts). Install the ATF strainer with the two bolts in the bolt holes (A) in step 12, and install the baffle plate with the two bolts in the bolt holes (B) in step 2 in shaft assemblies and housing installation.
  1. Install the regulator separator plate and two dowel pins on the main valve body.
  1. Install the new O-ring on the stator shaft, and install the stator shaft and stator shaft stop.
  1. Install the regulator valve body (eight bolts).
  1. Install the new O-ring on the ATF strainer, and install the ATF strainer (two bolts).
  1. Install the ATF joint pipes (one bolt).
  1. Install the ATF feed pipes in the regulator valve body and servo body.
