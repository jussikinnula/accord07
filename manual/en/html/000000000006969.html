Troubleshooting the CAN circuit related problems

Using the HDS (Preferred method)

  1. Go to B-CAN system diagnosis test mode A.
  1. If no DTCs are retrieved, go to Test Mode C or D.

Without the HDS (Should be used if the HDS is unavailable)

  1. Check for communication circuit problems using Mode 1.
  1. Check for DTCs while in Mode A.
  1. Sort, and then troubleshoot the DTCs in the order. Refer to the DTC troubleshooting for DTC descriptions.
    1st.
    Internal Error DTCs
    2nd.
    Loss of communication DTCs
    3rd.
    Signal Error DTCs

  1. If no DTCs are retrieved, use mode 2 to check all inputs related to failure.
NOTE: Not all inputs apply to all trim levels.
