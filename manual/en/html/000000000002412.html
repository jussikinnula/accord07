How to Troubleshoot Power Tailgate Control System
Power tailgate control unit has the self-diagnosis function that can detect failure without HDS.
  • If the power tailgate system has a failure, the system will initiate fail-safe mode, and ignition switch turned ON (II), the power tailgate indicator will start to blink.
  • In case the ignition switch is turned ON (II) again, if the tailgate system detects the same reoccurring failure, the system will initiate fail-safe mode and the power tailgate indicator will come on (no blinking).
How to troubleshooting
  1. Turn the ignition switch ON (II).
  1. Connect the SCS connector (A) to the power tailgate inspection connector (B).
    • If the ignition switch is turned OFF or SCS connector is disconnected while the safe-diagonals function is working, the safe-diagnosis function stops.
    • If the power tailgate control unit does not memorize DTC, the power tailgate indicator turns on for 3 seconds, then the self-diagnosis is complete.
  1. The power tailgate indicator displays DTCs in occurrence order while system is in self-diagnosis mode.

How to exit self-diagnosis mode
  1. Connect HDS to Data Link Connector (DLC).
  1. Clear the DTCs by selecting the DTCs MENU (DTCs).
