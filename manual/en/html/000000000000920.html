A/T Countershaft Speed Sensor Replacement
  1. Disconnect the output shaft (countershaft) speed sensor connector, and remove the output shaft (countershaft) speed sensor.
  1. Install the new O-ring (A) on the new output shaft (countershaft) speed sensor (B), then install the output shaft (countershaft) speed sensor in the transmission housing.
  1. Check the connector for rust, dirt, or oil, then connect the connector securely.
