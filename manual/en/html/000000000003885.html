MICS Troubleshooting - Test Mode D
B-CAN System Diagnosis Test Mode D
NOTE:
  • Perform this diagnosis if a component that is controlled by the B-CAN System does not run or does not come on.
  • If the component does not turn off or does not stop, go to B-CAN System diagnosis test mode D.
  • See the B-CAN System unit input/output index for a list of input and output devices and the control units that monitor the input and control the output devices.
  • Always cycle the ignition switch within 3 seconds when prompted in the DTC troubleshooting procedures in this system.
  1. Check the fuse of the malfunctioning output device.
Is the fuse OK?
YES -
Go to Step 2 .
NO -
Replace the fuse and recheck.■
  1. Check for DTCs by selecting the TEST MODE MENU from the Honda Diagnostic System (HDS).
Are any DTCs indicated ?
YES -
Go to B-CAN System diagnosis test mode A.■
NO -
Go to Step 3 .
  1. Turn ON the switch that controls the malfunctioning componention.
  1. Select DATA LIST from the TEST MODE MENU, and check the output signal for the malfunctioning component.
Is there an output signal?
YES -
Go to Step 5 .
NO -
Go to Step 9 .
  1. Check the relay and ground, then check for an open or a short in the circuit for the malfunctioning component.
Are the relay and circuit OK?
YES -

  1. Perform the function test for the malfunctioning component.
Does the output device pass the function test?
YES -

NO -

  1. With the malfunctioning connected, voltmeter between the malfunctioning output device and body ground on the wire that the control unit uses to control the component.
  1. Select the MISC. TEST of the TEST MODE MENU, and do the forced operation test of the malfunctioning component.
