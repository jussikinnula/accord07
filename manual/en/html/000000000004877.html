LKAS Camera Aiming Failure Codes
Aiming failure when the vehicle is stopped
NOTE:The aiming failure code resembles when the aiming procedure fails and is displayed on the HDS. A different code is indicated according to the vehicle condition in the aiming failure code.
Static camera aiming failure code indicated on the HDS when the vehicle is stopped.
Static Camera Aiming Failure Code (Screen Display)
Detection Item
Diagnostic Procedure
Camera aiming failure code 1
LKAS camera detected trouble.
  • When DTC is detected
  • The DTC is displayed by the HDS, and then performs the troubleshooting which is displayed. (DTC for aiming should be displayed before turn the ignition switch OFF).
  • After the DTC is cleared, do the aiming again.
Camera aiming failure code 2
  • The position of three targets (center, passenger's side and driver's side) is not correct. (Prescribed distance, height and distance between targets are not equal.)
After the following are confirmed, do the aiming again.
  • Confirm the prescribed distance.
  • Confirm the height.
  • Confirm the distance between targets.
Camera aiming failure code 3
The camera attitude-angle exceeds the range which is possible to aim.
  • The vehicle is not performing the correct position to the target.
  • The vehicle is not horizontally set up.
  • The arrangement of three targets (center, passenger's side and driver's side) is not correct.
  • The LKAS is not correctly installed.
  • There is luggage in the trunk of the car.
  • The air pressure in the tyre is not correctly adjusted.
  • The vehicle inclined abnormally.(Remodeling and the abrasion of the suspension etc).
After the following are confirmed, perform the aiming again.
  • The vehicle and the target are correctly set up.
  • The LKAS camera is correctly installed.
  • There must not be other loads than specified concerning the trunk.
  • The air pressure in the tyre is correctly adjusted.
  • There must be neither remodeling nor abrasion of the suspension. If it is necessary, perform the wheel alignment adjustment.
Camera aiming failure code 4
The central target cannot be detected.
  • The front glass or the LKAS camera lens is dirty.
  • Lack of light or the back light.
  • There are bright areas such as windows, illumination, light behind the target.
  • Something is placed on the instrument panel and is reflected in the glass.
  • The arrangement of the target is not correct.
  • LKAS camera malfunction.
After the following are confirmed, perform the aiming again.
  • Confirm the prescribed distance.
  • Confirm the height.
  • Confirm the distance between targets.
  • Arrange a condition to look clear the target.
Camera aiming failure code 5
The passenger's side target can not be detected.
  • The left target was not able to be detected.
Camera aiming failure code 6
The driver's side target can not be detected.
  • The right target could not be detected.

Aiming failure when driving the vehicle
NOTE: The aiming failure code resembles when the aiming procedure fails and is displayed on the HDS. A different code is indicated according to the vehicle condition in the aiming failure code.
Dynamic camera aiming failure code (1, 2, or 3) indicated on the HDS while the vehicle is driving.
Code
Detection Item
Diagnostic Procedure
1
LKAS Camera failure
Check the LKAS camera DTCs with the HDS, and perform the appropriate troubleshooting for the code indicated.
After DTC is cleared, it does an aiming procedure first.
2
LKAS Camera angle malposition
After the following are confirmed, aiming is performed again.
  • The vehicle and the target are correctly set up.
  • The LKAS camera is correctly installed.
  • There must not be other loads than specified concerning the trunk.
  • The air pressure in the tyres is correctly adjusted.
  • There must be neither remodeling nor any attenuation etc of the suspension. If it is necessary, the wheel alignment needs to be adjusted.
3
LKAS control unit failure
Check the LKAS control unit DTCs with the HDS, then perform the appropriate troubleshooting for the code indicated.
After the DTC is cleared, it does an aiming procedure first.
