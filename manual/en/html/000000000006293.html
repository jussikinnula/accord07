A/T Clutch Clearance Inspection
Special Tools Required

Clutch compressor attachment
 07ZAE-PRP0100
  1. Inspect the clutch piston, discs, plates, and end-plate for wear and damage, and inspect clutch wave-plate phase difference, if necessary.
  1. Install the clutch piston in the clutch drum. Do not install the O-rings during inspection.
  1. For K20Z2 engine model 1st clutch:
    Install the waved spring (A) in the clutch drum (B). Starting with the clutch wave-plate, alternately install the clutch wave-plates (C) (4), and discs (D) (4), and install the clutch end-plate (E) with the flat side toward the disc.
  1. For K20Z2 engine model 2nd clutch:
    Install the waved spring (A) in the clutch drum (B). Install the clutch flat-plate (C), and clutch disc (D). Starting with the clutch wave-plate, alternately install the clutch wave-plates (E) (3), and discs (F) (3), and install the clutch end-plate (G) with the flat side toward the disc.
  1. For K20Z2 engine model 3rd clutch:
    Install the waved spring (A) in the clutch drum (B). Install the clutch flat-plate (C), and clutch disc (D). Starting with the clutch wave-plate, alternately install the clutch wave-plates (E) (3), and discs (F) (3), and install the clutch end-plate (G) with the flat side toward the disc.
  1. For K20Z2 engine model 4th clutch:
    Install the waved spring (A) in the clutch drum (B). Starting with the clutch wave-plate, alternately install the clutch wave-plates (C) (3), and discs (D) (3), and install the clutch end-plate (E) with the flat side toward the disc.
  1. For K20Z2 engine model 5th clutch:
    Install the waved spring (A) in the clutch drum (B). Starting with the clutch wave-plate, alternately install the clutch wave-plates (C) (3), and discs (D) (3), and install the clutch end-plate (E) with the flat side toward the disc.
  1. For K24A3 engine model 1st clutch:
    Install the waved spring (A) in the clutch drum (B). Starting with the clutch plate, alternately install the clutch plate (C) (5) and discs (D) (5), then install the clutch end-plate (E) with the flat side toward the disc.
  1. For K24A3 engine model 2nd clutch:
    Install the waved spring (A) in the 2nd clutch drum (B). Install the clutch flat-plate (C), then starting with the clutch disc, alternately install the discs (D) (6) and clutch wave-plates (E) (5). Install the clutch end-plate (F) with the flat side toward the disc.
  1. For K24A3 engine model 3rd clutch:
    Install the waved spring (A) in the clutch drum (B). Starting with the clutch plate, alternately install the clutch plate (C) (6) and discs (D) (6), then install the clutch end-plate (E) with the flat side toward the disc.
  1. For K24A3 engine model 4th clutch:
    Install the waved spring (A) in the 4th clutch drum (B). Starting with the clutch wave-plate, alternately install the wave-plates (C) (4) and discs (D) (4). Install the clutch end-plate (E) with the flat side toward the disc.
  1. For K24A3 engine model 5th clutch:
    Install the waved spring (A) in the 5th clutch drum (B). Starting with the clutch wave-plate, alternately install the wave-plates (C) (4) and discs (D) (4). Install the clutch end-plate (E) with the flat side toward the disc.
  1. Install the snap ring with a screwdriver.
  1. Set a dial indicator (A) on the clutch end-plate (B).
  1. Zero the dial indicator with the clutch end-plate lifted up to the snap ring (C).
  1. Release the clutch end-plate to lower the clutch end-plate, then put the clutch compressor attachment on the end-plate (A).
  1. Except for K24A3 engine model 1st and 3rd clutches: Press the clutch compressor attachment down with 150−160 N (15−16 kgf, 33−35 lbf) using a force gauge, and read the dial indicator (B).
  1. For K24A3 engine model 1st and 3rd clutches:
    Press the clutch compressor attachment down with 39 N (4.0 kgf, 9 lbf) using a force gauge, and read the dial indicator (B).
  1. The dial indicator reads the clearance (C) between the clutch end-plate and top disc (D). Take measurements in at least three places, and use average as the actual clearance.
    Clutch End-Plate-to-Top Disc Clearance
    K20Z2 Engine Model:
    Service Limit:
    1st Clutch:
    1.23−1.43 mm (0.048−0.056 in.)
    2nd Clutch:
    0.75−0.95 mm (0.030−0.037 in.)
    3rd Clutch:
    0.83−1.03 mm (0.033−0.041 in.)
    4th Clutch:
    0.73−0.93 mm (0.029−0.037 in.)
    5th Clutch:
    0.73−0.93 mm (0.029−0.037 in.)

    K24A3 Engine Model:
    Service Limit:
    1st Clutch:
    0.82−1.02 mm (0.032−0.040 in.)
    2nd Clutch:
    1.14−1.34 mm (0.045−0.053 in.)
    3rd Clutch:
    1.00−1.20 mm (0.039−0.047 in.)
    4th Clutch:
    0.93−1.10 mm (0.037−0.044 in.)
    5th Clutch:
    0.93−1.10 mm (0.037−0.044 in.)

  1. If the clearance is out of the service limit, select a new clutch end-plate from the following table.
