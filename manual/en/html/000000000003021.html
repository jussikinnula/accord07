M/T Countershaft Reassembly

Special Tools Required

NOTE: Refer to the Exploded View as needed during this procedure.
  1. Clean all parts in solvent, dry them, and apply lubricant to all contact surfaces.
  1. Install the countershaft reverse gear (A), needle bearing (B), and 1st gear (C) onto the countershaft (D).
  1. Install the triple cone synchro assembly (A) by aligning the synchro cone fingers (B) with the holes in 1st gear (C), then install the synchro spring (D).
  1. Install the 1st/2nd synchro hub (A) by aligning the outer synchro ring fingers (B) with grooves in 1st/2nd synchro hub (C).
  1. Install the 1st/2nd synchro sleeve.
  1. Install the synchro spring (A).
  1. Install the triple cone synchro assembly (B) by aligning the outer synchro ring fingers (C) with the grooves in 1st/2nd synchro hub (D).
  1. Install the distance collar and needle bearing.
  1. Install the 2nd gear (A) by aligning the synchro cone fingers (B) with the holes in 2nd gear (C).
  1. Support the countershaft (A) on the steel blocks, then install the 3rd gear (B) using the special tool and a press (C).
  1. Install the 4th gear (A) using the special tool and a press (B).
  1. Install the 5th gear (A) using the special tool and a press (B).
  1. Install the distance collar (A) using the special tool and a press (B).
  1. Install the 35 mm shim (A) and the old ball bearing (B) using the special tool and a press (C).
  1. Measure the clearance between the old bearing (A) and the 35 mm shim (B) with a feeler gauge (C).
    Standard:
    0.04−0.10 mm (0.0016−0.0039 in.)

  1. If the clearance is more than the standard, select a new shim from the following table .

  1. Support the ball bearing (A) on steel blocks (B), then using a press (C) and an attachment (D) to press the countershaft out of the old ball bearing.
  1. If a new 35 mm shim was selected in step 16, install the shim and recheck the clearance.
  1. Install the distance collar (A) using the special tool and a press (B).
  1. Install the 35 mm shim (A) and new bearing (B) using the special tool and a press (C).
  1. Securely clamp the countershaft assembly in a bench vise with wood blocks (A).
  1. Tighten the new 20 mm special bolt (B) (left-hand threads).
