Cylinder Head Valve, Spring, and Valve Seal Installation
Special Tools Required

Stem seal driver
 07PAD-0010000

Valve spring compressor
 07757-0010000
  1. Coat the valve stems with engine oil. Install the valves in the valve guides.
  1. Check that the valves move up and down smoothly.
  1. Install the spring seats on the cylinder head.
  1. Install the new valve seals (A) using the valve guide seal installer (B).
    NOTE: The exhaust valve seal (C) has a black spring (D), and the intake valve seal (E) has a white spring (F). They are not interchangeable.
  1. Install the valve springs.
  1. Install the valve retainer.
    NOTE:
    • Before installing valve retainers and valve cotters, make sure both the shape of valve retainers and the color of valve cotters.
    • If you have SILVER cotters, you must use the valve retainers which have the flat shape of the round. If you have BLACK cotters, you must use the tapered shape of the round as shown.
    • Do not mix SILVER and BLACK cotters.
Surely install with a SILVER cotter
Surely install with a BLACK cotter
  1. Install the valve spring compressor. Compress the spring, and install the valve cotters.
  1. Remove the valve spring compressor.
  1. Lightly tap the end of each valve stem two or three times with a plastic mallet (A) to ensure proper seating of the valve and valve cotters. Tap the valve stem only along its axis so you do not bend the stem.
