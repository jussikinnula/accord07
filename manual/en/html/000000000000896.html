A/T End Cover ATF Feed Pipe Replacement
  1. Remove the snap rings (A), ATF feed pipes (B), and feed pipe flanges (C) from the end cover (D).
    NOTE: Replace the end cover, if the 1st clutch ATF feed pipe (E) replacement is required.
  1. Install the new O-rings (F) over the ATF feed pipes.
  1. Install the ATF feed pipes in the end cover by aligning the feed pipe tabs with the indentations in the end cover.
  1. Install the new O-rings (G) in the end cover, then install the feed pipe flanges over the ATF feed pipes.
  1. Secure the ATF feed pipes and feed pipe flanges with the snap rings.
