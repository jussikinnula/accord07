MICS Troubleshooting - Test Mode A
B-CAN System Diagnosis Test Mode A
NOTE:
  • Perform this diagnosis first if the symptom is related to the B-CAN system.
  • Because several different tester's can be used to troubleshoot, there might be differences in what is shown on the screen and what is shown in the DTC troubleshooting flow charts in the Shop Manual.
  • Always cycle the ignition switch within 3 seconds when prompted in the DTC troubleshooting procedures.
  1. Make sure the system is related to B-CAN.
    • Gauge control module
    • Exterior lights
    • Turn signals
    • Entry light control
    • Interior lights
    • Safety indicators
    • Rear window defogger (climate control)
    • Horns (security)
    • Chimes (key and lights ON)
    • Power windows
    • Wiper/washer
    • Security
    • Keyless entry
    • Power door locks
    • Climate control
    • Key interlock
    • Dash light brightness
    • Power tailgate control unit
Is the symptom related to the B-CAN system?
YES -
Go to Step 2 .
NO -
Go to the system troubleshooting for the system with the symptom.■
  1. Connect the Honda Diagnostic System (HDS).
  1. From BODY ELECTRICAL MENU, select the UNIT INFORMATION MENU, and then select CONNECTED UNIT.
    • Multiplex integrated control unit (MICU)
    • Door multiplex control unit
    • Gauge control module
    • Combination switch control unit
    • Relay control module
    • Climate control unit
    • Power tailgate control unit
    NOTE:
    • If the unit is communicating with the HDS, DETECT will be displayed.
    • If the unit is not communicating, N/A will be displayed.
Are all control units communicating with the HDS?
YES -
Go to Step 4 .
NO -
If any of them are not communicating, go to B-CAN System Dignosis Test B. If all units are not communicating, go to DTC B1000 troubleshooting.■
  1. Select the system that has the problem from the BODY ELECTRICAL MENU, then select DTCs.
Are any DTCs indicated?
YES -
Go to Step 5 .
NO -
If the problem is related to the following, go to B-CAN system diagnosis test mode C.
  • Exterior lights
  • Turn signals
  • Entry light control
  • Interior lights
  • Rear window defogger (climate control)
  • Horns (security)
  • Power windows
  • Wiper/washer
  • Power door locks
NOTE: In the problem is related to the following, go to the troubleshooting for the individual system.■
  • Gauge control module
  • Safety indicators
  • Chimes (key and lights ON)
  • Security
  • Keyless entry
  • Climate control
  • Key interlock
  • Dash lights blightness
  • Power tailgate control unit
  1. Record all DTCs and sort them by DTC type using the DTC troubleshooting.
  1. Troubleshoot the DTC(s) in the following order:
    • Internal error DTCs.
    • Loss of communication DTCs. Begin troubleshooting with the lowest number first (Example: If DTC B1006 and B1058 are retrieved, begin by troubleshooting B1006).
