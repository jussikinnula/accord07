DTC Troubleshooting: 28
LKAS Control Unit DTC 28:
 Wheel Speed Signal Value Error (Wheel Speed Signal)
  1. Clear the DTCs with the HDS.
  1. Do the following check.
    • Suspension is not modified
    • All four tyres the correct size, same manufacturer, and same type
    • All four tyres the correct pressure
Are they normally?
YES -
Go to Step 3 .
NO -
  • Adjust the wheel alignment
  • Adjust the tyre pressure
  • Change for the standard tyres
  1. Start the engine and test-drive a car for two minutes or more every 65 km/h (41 mph) or more.
  1. Watch the adaptive cruise control (ACC)/LKAS indicator.
Does the adaptive cruise control (ACC)/LKAS indicator come on?
YES -
Go to Step 5 .
NO -
Intermittent failure, the system is OK at this time.■
  1. Check for DTC with HDS.
Is DTC 28 indicated?
YES -
Check for loose terminals in the LKAS control unit connectors, and repair if necessary. If no poor connections are found, replace the LKAS control unit. ■
NO -
Do the appropriate troubleshooting for the code indicated.■
