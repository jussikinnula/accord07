EPS Control Unit Removal/Installation
  1. Make sure you have the anti-theft code for the radio, then write down the frequencies for the radio preset buttons. Remove the battery.
  1. Remove the passenger's side kick panel.
  1. Disconnect the EPS control unit connectors (A).
  1. Remove the EPS control unit (B).
  1. Install the EPS control unit in the reverse order of removal.
    NOTE: If the EPS control unit is replaced, the EPS control unit must memorize the torque sensor neutral position.
  1. Reconnect the battery.
  1. Reset the power window control unit.
  1. Enter the anti-theft codes for the radio, then enter the customer's radio station presets. Reset the clock.
  1. After installation, start the engine, and let it idle. Turn the steering wheel from lock-to-lock several times. Check that the EPS indicator does not come on.
