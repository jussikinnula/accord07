A/T Clutch Reassembly
Special Tools Required

Clutch spring compressor set
 07LAE-PX40000

Clutch spring compressor attachment
  07LAE-PX40100

Clutch spring compressor attachment
  07HAE-PL50101

Clutch spring compressor bolt assembly
  07GAE-PG40200

Snap ring pliers
 07LGC-0010100
  1. Soak the clutch discs thoroughly in ATF for a minimum of 30 minutes.
  1. Install the new O-rings (A) on the 4th and 5th clutch piston (B).
  1. Install the new O-ring (A) in the 1st, 2nd, and 3rd clutch pistons (B), and install the new O-ring (C) on the clutch drums (D).
  1. Install the clutch piston (A) in the clutch drum (B). Apply pressure and rotate to ensure proper seating. Lubricate the piston O-ring with ATF before installing. Do not pinch the O-ring by installing the piston with to much force.
  1. Install the return spring (A) and spring retainer (B), and position the snap ring (C) on the retainer.
  1. Install the special tools.
  1. Be sure the special tool (A) is adjusted to have full contact with the spring retainer (B) on the 4th and 5th clutches.
  1. Set the special tool (A) on the spring retainer (B) of the 1st, 2nd, and 3rd clutches in such a way that the special tool compresses the clutch return spring (C).
  1. If either end of the special tool is set over an area of the spring retainer which is unsupported by the return spring, the retainer may be damaged.
  1. Compress the return spring.
  1. Install the snap ring with snap ring pliers.
  1. Remove the special tools.
  1. For K20Z2 engine model 1st clutch:
    Install the waved spring (A) in the clutch drum (B). Starting with the clutch waved-plate, alternately install the clutch waved-plates (C) (4), and discs (D) (4), and install the clutch end-plate (E) with the flat side toward the disc.
  1. For K20Z2 engine model 2nd clutch:
    Install the waved spring (A) in the clutch drum (B). Install the clutch flat-plate (C), and clutch disc (D). Starting with the clutch waved-plate, alternately install the clutch waved-plates (E) (3), and discs (F) (3), and install the clutch end-plate (G) with the flat side toward the disc.
  1. For K20Z2 engine model 3rd clutch:
    Install the waved spring (A) in the clutch drum (B). Install the clutch flat-plate (C), and clutch disc (D). Starting with the clutch waved-plate, alternately install the clutch waved-plates (E) (3), and discs (F) (3), and install the clutch end-plate (G) with the flat side toward the disc.
  1. For K20Z2 engine model 4th clutch:
    Install the waved spring (A) in the clutch drum (B). Starting with the clutch waved-plate, alternately install the clutch waved-plates (C) (3), and discs (D) (3), and install the clutch end-plate (E) with the flat side toward the disc.
  1. For K20Z2 engine model 5th clutch:
    Install the waved spring (A) in the clutch drum (B). Starting with the clutch waved-plate, alternately install the clutch waved-plates (C) (3), and discs (D) (3), and install the clutch end-plate (E) with the flat side toward the disc.
  1. For K24A3 engine model 1st clutch:
    Install the waved spring (A) in the clutch drum (B). Starting with the clutch plate, alternately install the clutch plate (C) (5) and discs (D) (5), then install the clutch end-plate (E) with the flat side toward the disc.
  1. K24A3 engine model 2nd clutch:
    Install the waved spring (A) in the 2nd clutch drum (B). Install the clutch flat-plate (C), then starting with the clutch disc, alternately install the discs (D) (6) and clutch waved-plates (E) (5). Install the clutch end-plate (F) with the flat side toward the disc.
  1. For K24A3 engine model 3rd clutch:
    Install the waved spring (A) in the clutch drum (B). Starting with the clutch plate, alternately install the clutch plate (C) (6) and discs (D) (6), then install the clutch end-plate (E) with the flat side toward the disc.
  1. K24A3 engine model 4th clutch:
    Install the waved spring (A) in the 4th clutch drum (B). Starting with the clutch waved-plate, alternately install the waved-plates (C) (4) and discs (D) (4). Install the clutch end-plate (E) with the flat side toward the disc.
  1. K24A3 engine model 5th clutch:
    Install the waved spring (A) in the 5th clutch drum (B). Starting with the clutch waved-plate, alternately install the waved-plates (C) (4) and discs (D) (4). Install the clutch end-plate (E) with the flat side toward the disc.
  1. Install the snap ring with a screwdriver.
  1. Check that the clutch piston moves by applying air pressure into fluid passage.
