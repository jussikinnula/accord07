Headlight Adjuster Control Unit Input Test
With HID Lamp
  1. Remove the dashboard lower cover.
  1. Disconnect the 14P connecter (A) from the control unit (B).
  1. Inspect the connector and socket terminals to be sure they are all making good contact.
    • If the terminals are bent, loose or corroded, repair them as necessary, and recheck the system.
    • If the terminals look OK, make the following input tests at the connector.
      • -
      If a test indicates a problem, find and correct the cause, then recheck the system.
      -
      If all the input tests prove OK, the control unit must be faulty, replace it.
