Throttle Cable Adjustment
  1. K24A3 engine:
    Remove the throttle cable cover (A).
  1. Check cable free play at the throttle linkage. Cable free pley (A) should be 15−18 mm (9/16−11/16 in.) (K20A6 engine: 10−12 mm (3/8−1/2 in.)).
  1. If the free play is not within spec 15−18 mm (9/16−11/16 in.) (K20A6 engine: 10−12 mm, (3/8−1/2 in.)), loosen the locknut (B), turn the adjusting nut (C) until the deflection is as specified, then retighten the locknut.
  1. K20A6 engine:
    With the cable properly adjusted, check the throttle valve to be sure it opens fully when you push the accelerator pedal to the floor. Also check the throttle valve to be sure it returns to the idle position whenever you release the accelerator pedal.
  1. K23A3 engine:
    With the cable properly adjusted, check the throttle link to be sure it opens fully when you push the accelerator pedal to the floor. Also check the throttle link to be sure it returns whenever you release the accelerator pedal.
