A/T Assembly Installation
  1. Flush the ATF cooler.
  1. Install the torque converter assembly (A) on the mainshaft (B) with the new O-ring (C).
  1. Install the 14 x 20 mm dowel pins (D) in the torque converter housing.
  1. Place the transmission on the jack, and raise the transmission to the engine level.
  1. Attach the transmission to the engine, then install the transmission housing mounting bolts.
  1. Install the transmission housing mounting bolts.
  1. Install the transmission housing mounting bolts.
  1. Remove the jack from the transmission.
  1. Install the front mount bracket with the new bolts.
  1. Install the rear mount bracket (A).
  1. Install the rear mount/base bracket (B).
  1. Install the new set ring (A) on the intermediate shaft (B).
  1. Clean the areas where the intermediate shaft contacts the transmission (differential) with solvent or carburetor cleaner, and dry with compressed air. Then install the intermediate shaft (B), be sure not to allow dust or other foreign particles to enter the transmission.
  1. Install the new set ring on the left driveshaft.
  1. Clean the areas where the left driveshaft contacts the transmission (differential) with solvent or carburetor cleaner, and dry with compressed air. Then install the left driveshaft, be sure not to allow dust or other foreign particles to enter the transmission. Turn the steering knuckle fully outward, and slide the driveshaft into the differential until you feel its set ring engage the side gear.
  1. Slide the right driveshaft over the intermediate shaft splines until you feel the driveshaft engage the intermediate shaft set ring.
  1. Install the transmission lower mounts.
  1. Support the front subframe with a wood piece and a jack, and lift it up to body.
  1. Slide the steering gearbox (A) in the gearbox mounting bracket (B). Apply a neutral detergent (diluted with water) to the steering gearbox bushing (C) surfaces upper and lower. If you don't apply a neutral detergent to the bushing surfaces, the gearbox bushing will be damaged when installing the subframe.
  1. Loosely install the new subframe mounting bolts (D), stiffener mounting bolts (E), and new stiffener mounting bolts (F).
  1. Loosely install both of the new mid-mounts mounting bolts.
  1. Align the reference marks (A) with edge (B) of both rear stiffeners (C), and tighten the rear subframe mounting bolts, then front bolts, and tighten the stiffener bolts to the specified torque.
  1. Tighten the mid-mount mounting bolts.
  1. Install the rear mount base bracket bolts.
  1. Install the transmission lower mount nuts.
  1. Install the steering gearbox mounting bolts, and install new stiffener bolt.
    NOTE: The illustration shows LHD model; RHD model is similar.
  1. Install the steering gearbox mounting bracket bolts.
    NOTE: The illustration shows LHD model; RHD model is similar.
  1. Secure the power steering fluid line bracket (hydraulic P/S model) or harness clamp bracket (EPS model) on the subframe with the bolt.
    NOTE: The illustration shows LHD model; RHD model is similar.
  1. Install the rear mount stop.
  1. Attach the shift cable end (A) to the control lever (B), then insert the control pin (C) into the control lever hole through the shift cable end, and secure the control pin with the spring clip (D). Do not bend the shift cable excessively.
  1. Install the shift cable cover (E), then secure the shift cable holder (F) to the shift cable cover with the three bolts.
  1. Attach the torque converter to the drive plate with eight bolts (A). Rotate the crankshaft pulley as necessary to tighten the bolts to 1/2 of the specified torque, then to the final torque, in a crisscross pattern. After tightening the last bolt, check that the crankshaft rotates freely.
  1. Install the engine stiffener (B).
  1. Secure the power steering fluid line with the clamps (hydraulic P/S model).
  1. Secure the power steering fluid line bracket (A) with the bolt (hydraulic P/S model). Install the steering gearbox heat shield (B).
    NOTE: The illustration shows LHD model; RHD model is similar.
  1. Install the exhaust pipe A with new gaskets (B).
  1. Install the damper forks (A) with damper pinch bolts (B) and new self-locking nuts (C), then install the ball joints on the lower arms (D) with the castle nuts (E) and new cotter pins (F).
    NOTE: Be careful not to damage the headlight adjuster leveling sensor arm.
  1. Install the stabilizer links (G) to the lower arms, and install the nuts (H). Insert a 6 mm Allen wrench (I) in the ball joint pin (J), and tighten the nuts.
  1. Install the new front mount bolt (A).
  1. Install the front mount stop (B) and vacuum hose clamp bracket (C).
  1. Connect the vacuum hose (D) to the vacuum line (E), then install the hose on its clamp (F) at the mark (G).
  1. Hold the stud nut (A) with a wrench, and remove the hanger plates and bolts. Tighten the EGR passage mounting bolts (three bolts) (B) and stud nut to the specified torque.
  1. Install the transmission upper mount bracket (A), transmission upper mount bracket plate (B), and ground cable (C).
  1. Connect the ATF cooler hose (A) to the ATF cooler line (B), and secure it with the clip.
  1. Install the ATF cooler hose (C) on the clamp (D) at the mark (E).
  1. Connect the ATF cooler hoses (A) to the ATF cooler lines (B), and secure the hoses with the clips.
  1. Connect the 3rd clutch transmission fluid pressure switch connector (A), and install the harness clamp (B) on the clamp bracket (C).
  1. Connect the shift solenoid harness connector (A), A/T clutch pressure control solenoid valve B (B) connector, and solenoid valve C (C) connector, and install the harness clamp (D) on the clamp bracket (E).
  1. Connect the countershaft speed sensor connector (A) and mainshaft speed sensor connector (B).
  1. Connect the A/F sensor connector (C), then install it on its bracket (D).
  1. Connect the transmission range switch connector (E), then install it on its bracket (F).
  1. Install the harness clamps (G) on the harness clamp brackets (H).
  1. Install the A/T clutch pressure control solenoid valve A connector and 2nd clutch transmission fluid pressure switch connector (B), and install the harness clamps (C) on the harness clamp brackets (D).
  1. Refill the transmission with ATF.
  1. Install the battery base.
  1. Install the air cleaner housing and intake air duct.
  1. Install the battery tray, battery, and battery cover, then secure the battery with its hold-down bracket.
  1. Connect the battery terminals.
  1. Set the parking brake. Start the engine, and shift the transmission through all gears three times.
  1. Check the shift lever operation, A/T gear position indicator operation, and shift cable adjustment.
  1. Check and adjust the front wheel alignment.
  1. Install the splash shield.
  1. Turn off the engine, and check the ATF level.
  1. Do the power window control unit reset procedure.
  1. Do the road test.
  1. Enter the anti-theft code for the radio (and navigation system), then enter the radio station presets, and set the clock.
