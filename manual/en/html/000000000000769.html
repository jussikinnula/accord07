M/T Assembly Installation
  1. Check that the two dowel pins are installed in the clutch housing.
  1. Apply urea grease UM264 (P/N 41211-PY5-305) to the release fork (A) and the release bearing (B). Install the release fork and the release bearing.
  1. Install the front lower transmission mount (A) and rear lower transmission mount (B).
  1. Place the transmission on the transmission jack, and raise it to the engine level.
  1. Install the lower transmission mounting bolts.
  1. Install the clutch cover.

6-speed model:
  1. Install the upper rear engine mount bracket with the new bolts.
  1. Install the rear engine mount (A), upper rear engine mount mounting bolt (B), and shift cable bracket (C).
  1. Install the upper front engine mount bracket with the new bolts.
  1. Support the subframe with a 4 x 4 x 40 in. piece of wood and a jack.
  1. Align the reference marks (A) with edge (B) of both rear stiffer (C), and tighten the rear subframe mounting bolts, then front bolts, and tighten the stiffener bolts to the specified torque.
  1. Install the front suspension subframe (A) and front suspension subframe stays (B).
  1. Install the middle subframe mounting bolts.
  1. Install the lower transmission mount mounting nuts.
  1. Install the lower rear engine mount mounting bolts.
  1. Install the oil pipe.
  1. Install the steering line bracket.
  1. Install the heat shield.
  1. Install the intermediate shaft.
  1. Install the driveshafts.
  1. Install the exhaust pipe A.
  1. Install the splash shield.
  1. Install the transmission mount bracket (A) and ground cable (B).
  1. Install the upper transmission mounting bolts.
  1. Install the steering gearbox mounting bolts.
  1. Install the rear engine mount stop.
  1. Install the front engine mount mounting bolt (A) and front engine mount stop (B).
  1. Remove the chain hoist (A).
  1. Fix the long nut (B), using an appropriate tool.
  1. Remove the engine hanger (C) and bolt (D).
  1. Check the torque of EGR passage bolts (four places).
    Tightening Torque:
     22 N·m (2.2 kgf·m, 16 lbf·ft)

  1. Install the bracket (A), then connect the secondary heated oxygen sensor (secondary HO2S) connector (B).
  1. Connect the reverse lockout solenoid connector (A), back-up light switch connector (B), and countershaft speed sensor connector (C).
  1. Install the cable bracket (A) and cables (B).
  1. Apply a light coat of urea grease UM264 (P/N 41211-PY5-305) to the cable ends, and install new cotter pins (C).
  1. Apply urea grease UM264 (P/N 41211-PY5-305) to the end of the cylinder rod. Install the slave cylinder. Take care not to bend the clutch line.
  1. Install the battery tray.
  1. Install the air cleaner housing.
  1. Install the battery. Connect the positive (+) cable first, then the negative (−) cable to the battery.
  1. Refill the transmission fluid.
  1. Test-drive the vehicle.
  1. Check the clutch operation.
  1. Check the front wheel alignment.
  1. Enter the anti-theft code for the radio and navigation system, then enter the customer's radio station presets.
  1. Perform the power window control unit reset procedure.
