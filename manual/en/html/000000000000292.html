Rain Sensor Input Test
NOTE: When replacing the rain sensor, always install the rain sensor first, then connect the battery.
  1. Before testing the rain sensor, perform the multiplex integrated control system test mode A troubleshooting.
  1. Remove the cover (A) from the rain sensor (B).
  1. Disconnect the 5P connector (C) from the rain sensor.
  1. Inspect the connector and socket terminals to be sure they are all making good contact.
    • if the terminals are bent, loose or corroded, repair them as necessary, and recheck the system.
    • if the terminals look OK, go to Step 5 .
  1. With the connector still disconnected, make these input tests at the connector.
    • if any test indicates a problem, find and correct the cause, then recheck the system.
    • If all the input tests prove OK, replace the rain sensor.
