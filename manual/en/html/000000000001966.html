Engine Installation
  1. Install the accessory brackets and tighten their bolts to the specified torques.
  1. Install the engine/transmission into position in the vehicle.
    Reinstall the mounting bolts/support nuts in the sequence given. Failure to follow this sequence may cause excessive noise and vibration, and reduce bushing life.
  1. Install the side engine mount bracket (A), then loosely tighten the new bolts/nut (B), and loosely tighten the bolt (C).
  1. Install the transmission upper mount (A), then loosely tighten the new nuts (B), and loosely tighten the new bolt (C).
  1. Install the transmission upper mount bracket (A), and tighten the new bolts, then loosely tighten the mounting bolt (B).
  1. Raise the vehicle on the hoist to full height.
  1. Install the lower torque rod (A), then tighten the new bolts (B), and loosely tighten the new bolt (C).
  1. Remove the chain hoist from the engine.
  1. Lower the vehicle on the hoist.
  1. Tighten the side engine mount mounting bolts/nut in the numbered sequence shown.
  1. Tighten the transmission upper mount/bracket mounting bolts/nut in the numbered sequence shown.
  1. Raise the vehicle on the hoist to full height.
  1. Tighten the lower torque rod mounting bolt.
  1. Tighten the two bolts securing the transmission lower mount.
  1. Lower the vehicle on the hoist.
  1. Install the upper torque rod (A), then tighten the new bolts in the numbered sequence shown.
  1. Install the ground cable (B).
  1. Raise the vehicle on the hoist to full height.
  1. Install exhaust pipe A (A). Use new gaskets (B) and new self locking nuts (C).
  1. Install a new set ring on the end of each driveshaft, then install the driveshafts. Make sure each ring ‘‘clicks'' into place in the differential and intermediate shaft.
  1. Connect the suspension lower arm ball joints.
  1. Install the damper forks.
  1. Connect the stabilizer links.
  1. Install the splash shield.
  1. Install the front tyres/wheels.
  1. Lower the vehicle on the hoist.
  1. Install the A/C compressor.
  1. Install the A/C condenser fan shroud (A), then connect the A/C condenser fan motor connector (B).
  1. Install the upper radiator hose (A) and lower radiator hose (B).
  1. Install the heater hoses.
  1. Install the expansion tank.
  1. Install the power steering (P/S) pump (A), and the P/S hose clamp (B).
  1. Install the drive belt.
  1. Push the engine wire harness connectors through the bulkhead, then install the grommet (A) and harness clamps (B).
  1. Connect the accelerator pedal position (APP) sensor connector (C).
  1. Connect the engine wire harness connectors (A).
  1. Install the harness clamp (B).
  1. Install the fuel return hose (A) and brake booster vacuum hose (B).
  1. Install the fuel feed hose.
  1. Install the intercooler outlet hose.
  1. Install the select cable and shift cable.
  1. Install the clutch line bracket.
  1. Install the clutch slave cylinder and clutch line bracket mounting nut.
  1. Install the intercooler inlet pipe.
  1. Place the fuse/relay box bracket, then install the harness clamp (A) to the body.
  1. Install the harness clamps (B) and harness connector (C) to the fuse/relay box bracket.
  1. Tighten the four bolts (A) securing the fuse/relay box bracket and tighten the bolt (B) securing the intercooler inlet pipe.
  1. Install the under-hood fuse/relay box (C) to the fuse/relay box bracket.
  1. Connect the harness connectors (A).
  1. Install the glow control module (B) and harness clamps (C), then install the battery cables (D).
  1. Install the resonator (A), then install the air intake cover (B).
  1. Loosely install the bolts (A), then install the battery base (B).
  1. Tighten the bolts securing the battery base.
  1. Install the ground cable stay (C), harness clamp (D), and harness stay mounting bolt (E).
  1. Install the air cleaner housing.
  1. Install the engine cover.
  1. Install the battery. Clean the battery posts and cable terminals with sandpaper, then assemble them and apply grease to prevent corrosion.
  1. Check that the transmission shifts into gear smoothly.
  1. Refill the engine with engine oil.
  1. Refill the transmission with fluid.
  1. Do the fuel system bleed procedure.
  1. Refill the radiator with engine coolant, and bleed air from the cooling system with the heater valve open.
  1. Do the ECM reset procedure.
  1. Do the power window control unit reset procedure.
  1. Inspect the idle speed.
  1. Enter the anti-theft codes for the radio and the navigation system, then enter the customer's radio station presets.
  1. Set the clock.
