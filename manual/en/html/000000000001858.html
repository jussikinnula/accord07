DTC Troubleshooting: B1050, B1100, B1150, B1200, B1250, B1300, B1350
DTC B1050, B1100, B1150, B1200, B1250, B1300, B1350:
 Communication Bus Line Error
  1. Clear the DTCs using the HDS.
  1. Cycle the ignition switch to OFF, and then back ON.
  1. Check the DTCs by selecting the DTCs MENU (DTCs) from the HDS.
Are DTC(s) B1050, B1100, B1150, B1200, B1250, B1300 or B1350 indicated?
YES -
Go to Step 4 .
NO -
Intermittent failure, the system is OK at this time.■
  1. Check the DTCs by selecting the DTCs MENU (DTCs) from the HDS.
Is DTC B1000 indicated?
YES -
Troubleshoot DTC B1000.■
NO -
If DTC B1050 indicated, replace the relay module. If DTC B1100 is indicated, replace the door multiplex control unit. If DTC B1150 is indicated, troubleshoot DTC B1150. If DTC B1200 is indicated, replace the climate control unit. If DTC B1250 is indicated, replace the combination switch control unit. If DTC B1300 is indicated, replace the Rain sensor. If DTC B1350 is indicated, replace the Power tailgate control unit.■
