Engine Assembly Installation
Special Tools Required

Sub hanger stay
 07MAK-PY30100

Engine tilt hanger set
 07KAK-SJ40101

Engine hanger stay
 07KAK-SJ40200
  1. Install the accessory brackets and tighten their bolts to the specified torques.
  1. Install the engine/transmission assembly into position in the vehicle, then remove the engine tilt hanger set from the engine.
    NOTE: Reinstall the mounting bolts/support nuts in the sequence given. Failure to follow this sequence may cause excessive noise and vibration, and reduce bushing life.
  1. A/T model: Tighten the front mount bolt (A), then install the front mount stop (B) and vacuum hose clamp bracket (C). Install vacuum hose (D).
  1. Tighten the rear mount bolt (A), then install the rear mount stop (B).
  1. Install the upper bracket (A), then tighten the bolts in the numbered sequence shown.
  1. Install the ground cable (B).
  1. M/T model: Loosen the transmission upper mount bolt (A), then install the transmission upper mount/bracket assembly (B), clutch line clamp bracket (C), and ground cable (D).
  1. A/T model: Install the transmission upper mount (A), transmission upper bracket (B), and ground cable (C), then loosely install the transmission upper bracket plate (D).
  1. M/T model: Tighten the transmission upper mount bolt.
  1. A/T model: Tighten the transmission upper mount bracket plate mounting bolts (A), then tighten the transmission upper mount bolt (B).
  1. Raise the hoist to full height.
  1. Tighten the nuts securing the transmission lower front mount and transmission lower rear mount.
  1. A/T model: Install the shift cable (A).
  1. A/T model: Install the shift cable cover (B), then tighten the shift cable holder (C).
  1. Install a new set ring on the end of each driveshaft, then install the driveshafts. Make sure each ring ‘‘clicks'' into place in the differential and intermediate shaft.
  1. Connect the suspension lower arm ball joints.
  1. Install the damper fork.
  1. Connect the stabilizer links.
  1. Install exhaust pipe A. Use new gaskets (B) and new self locking nuts (C).
  1. Install the splash shield.
  1. Install the front wheels.
  1. Lower the vehicle on the hoist.
  1. Check the quick connector (A) and set ring (B) for cracks or damage. If the connector and/or set ring are cracked or damaged, replace the connector.
  1. Make sure the set ring is in place inside to the quick connector. If the set ring is off the connector, replace the quick connector.
  1. Replace the O-ring (C) in the quick connector.
  1. Check the lock (D) for damage of deformation. If the lock is damaged or deformed, replace it. When installing the new lock to the connector, straight down along the groove.
  1. Clean the connecting surface of the radiator (E), then apply clean engine coolant around the connecting surface.
  1. Push down the lock (A), then push the quick connector (B) onto the radiator until you hear it click.
  1. Install the upper radiator hose (A) and heater hoses (B).
  1. A/T model: Install the automatic transmission fluid (ATF) cooler hoses.
  1. Install the A/C compressor.
  1. Install the power steering (P/S) pump (A), and the P/S hose clamp (B).
  1. Install the drive belt.
  1. Connect the air fuel ratio (A/F) sensor connector.
  1. M/T model: Install the select cable (A) and shift cable (B) with the plastic washers (C), washers (D), and new cotter pins (E).
  1. M/T model: Install the clutch slave cylinder (A) and clutch line bracket mounting bolt (B).
  1. Push the engine control module (ECM)/powertrain control module (PCM) connectors through the bulkhead, then install the grommet (A) and harness clamps (B).
  1. Connect the accelerator pedal position (APP) sensor connector (C).
LHD model
RHD model
  1. Connect the ECM/PCM connectors (A) and main wire harness connectors (B).
LHD model
RHD model
  1. Install the evaporative emission (EVAP) canister hose (A) and brake booster vacuum hose (B).
  1. Install the fuel feed hose.
  1. Connect the harness connector (A).
  1. Install the under-hood fuse/relay box (B), then install the battery cables (C).
  1. Install the battery base (A), then install the harness clamps (B), and harness bracket (C).
  1. Install the air cleaner housing.
  1. Install the intake air duct (A), then connect the intake air temperature (IAT) sensor connector (B), and install the vacuum hose (C) and breather pipe (D).
  1. Install the battery. Clean the battery posts and cable terminals with sandpaper, then assemble them and apply grease to prevent corrosion.
  1. A/T model: Move the shift lever to each gear, and verify that the A/T gear position indicator follows the transmission range switch.
  1. M/T model: Check that the transmission shifts into gear smoothly.
  1. Inspect for fuel leaks. Turn the ignition switch ON (II) (do not operate the starter) so the fuel pump runs for about 2 seconds and pressurizes the fuel line. Repeat this operation two or three times, then check for fuel leakage at any point in the fuel line.
  1. Refill the engine with engine oil.
  1. Refill the transmission with fluid:
    • Manual transmission
    • Automatic transmission
  1. Refill the radiator with engine coolant, and bleed air from the cooling system with the heater valve open.
  1. Do the ECM/PCM reset procedure.
  1. Do the crankshaft position (CKP) pattern clear/CKP pattern learn procedure.
  1. Do the power window control unit reset procedure.
  1. Inspect the idle speed.
  1. Inspect the ignition timing.
  1. Enter the anti-theft codes for the radio and the navigation system, then enter the customer's audio presets.
  1. Set the clock.
