  1. Install the 2nd gear, 1st gear, 5th gear, and 37 x 41 x 82.8 mm collar on the countershaft.
  1. Slide the 3rd gear (A) over the countershaft, and press it in place with the special tool and a press.
  1. Install the cotters, 35 x 47 x 7.8 mm collar, set ring, needle bearing, and 4th gear.
  1. Slide the reverse selector hub (A) over the countershaft, then press it in place with the special tool and a press.
    NOTE: Some reverse selector hubs are not press-fitted and can be installed without using the special tool and a press.
